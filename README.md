# ✈️ Expedivo – Travel Booking Platform

A modern full-stack travel booking web application built with **Next.js**, **Express.js**, **MongoDB**, and **Better Auth**.  
Users can explore destinations, book trips, and manage their profiles seamlessly.

---

## 🚀 Live Demo

- 🌐 Client: https://expedivo-client.vercel.app  
- ⚙️ Server: https://expedivo-server.vercel.app  

---

## 📌 Features

### 👤 Authentication
- Google Login (Better Auth)
- Secure JWT-based session handling
- Protected routes

### 🌍 Destinations
- View all travel destinations
- Featured destinations section
- Detailed destination pages

### 🧳 Booking System
- Book travel destinations
- View personal bookings
- Cancel bookings

### 👨‍💼 User Profile
- View profile information
- Update name, email, and profile image
- Avatar support

### 🎨 UI/UX
- Fully responsive design
- Modern UI with Tailwind CSS
- Clean and minimal layout

---

## 🛠️ Tech Stack

### Frontend
